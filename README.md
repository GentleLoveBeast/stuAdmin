# stuAdmin 信息管理系统
## server
  * node后台
  * 在server文件夹下输入命令 ` npm install ` 安装依赖
  * 启动后台时，在server文件夹下输入 ` node app.js ` 
  * 后台端口号为 3005
## src
  * vue前台
  * 在src文件夹下输入命令 ` npm install ` 安装依赖
  * 启动前台时，在src文件夹下输入 ` npm run dev ` 
  * 前台端口号为 3000，必须是 ` 127.0.0.1:3000/ ` ，不能 ` localhost:3000/ ` 
## database
  * mongodb数据库
  * 默认无数据，注册账户，添加信息插入数据
  * 两个集合的名字为 ` students ` 和 ` users `
